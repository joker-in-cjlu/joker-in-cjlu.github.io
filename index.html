SCA、SAST、DAST 是什么 缩写 全称 SCA Software Composition Analysis（软件成分分析） SAST Static Application Security Testing（静态应用程序安全测试） DAST Dynamic Application Security Testing（动态应用程序安全测试） 个人可以理解为： SCA：只检查每个组成部分（开源组件、第三方库文件），不检查代码漏洞和运行漏洞。 SAST：根据源码判断设计缺陷漏洞，判断源代码中的可能漏洞，如： 注入式漏洞：SQL 注入、命令注入 跨站脚本 XSS：反射型 XSS、存储型 XSS、基于 DOM 的 XSS 加密与随机数问题：使用 MD5、SHA1 等弱哈希算法，使用不安全的随机数生成器 代码质量：如空指针、内存资源分配、日志泄露、错误信息泄露（API、密码） DAST：检测实际运行时出现的漏洞（黑盒测试无需源码），包含配置环境、API 认证问题。 三种安全测试方法对比 SCA SAST DAST 全称 Software C...

中国计量大学 · 自动化专业学校背景中国计量大学是国内唯一一所以”计量”命名的本科院校，位于杭州下沙。学校的特色是计量、检测、质量控制，自动化专业也围绕这些方向发展。 专业定位自动化隶属机电工程学院，是国家一流本科专业建设点。最大特色是把自动控制与精密检测结合起来，毕业后既懂控制，也懂测量。 主要课程 类别 课程 数学基础 高等数学、线性代数、概率论与数理统计 专业核心 自动控制原理、现代控制理论、过程控制系统 检测特色 传感器技术、精密测量技术、智能仪器设计 硬件开发 微机原理与接口技术、嵌入式系统设计 软件工具 C 语言、Python、MATLAB/Simulink 实践环节 电子实习、PLC 实训、毕业设计 就业方向 领域 典型去向 计量检测 各省市计量院、质检院、SGS 智能制造 海康、大华、中控、汇川 汽车电子 吉利、零跑、博世 考研深造 浙大、杭电、本校控制/仪器方向 计量大学的自动化——和一般工科院校不同，多了”计量检测”这个独门标签，去质检、计量相关单位有明显的院校优势。

香港大学 IDT —— 创新设计与技术理学硕士 香港大学（The University of Hong Kong, HKU）是亚洲顶尖的研究型综合大学，其工程学院于 2022 年秋季新开设了创新设计与技术工程理学硕士（MSc in Innovative Design and Technology，简称 IDT）。这是一个由工学院、教育学院和建筑学院联合授课的跨学科项目，旨在培养能够用设计与技术驱动创新的复合型人才。 培养目标IDT 专为快速变化的数字化时代而设计。课程强调”技术 + 设计”双轮驱动，学生不仅要掌握前沿工程技术，还要具备以用户为中心的设计思维。通过跨学科的顶点项目（Capstone Project），学生将所学应用于解决真实世界的问题。 课程体系课程共 72 学分，包含三大板块： 创新设计核心课：涵盖机电系统设计、无人机技术、产品开发、人因工程、多媒体技术等 技术方向选修：提供人工智能与机器人、STEM 教育、智慧城市、自由定制四个方向 毕业项目：占 24 学分的 Capstone，是整个学位的核心 学生可以从 AI 与机器人、智慧城市、STEM 教育等热门领域...
